harmonica WordPress Theme
---------------------
harmonica is a parent theme for WordPress and harmonica child

Created by Hence (Han) Wijaya, https://wordpress.org/support/profile/hwijaya


Install
-------
1. Upload the harmonica folder via FTP to your wp-content/themes/ directory.
2. Go to your WordPress dashboard and select Appearance.
3. Be sure to activate the harmonica child theme, and not the harmonica parent theme.
